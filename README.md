# Парсинг контактных данных с сайта Zoon

Удалось спарсить более 10000 контактов. 

Библиотеки: 
1. BeautifulSoup
2. requests
3. selenium

Задача заключалась в том, чтобы показать навыки работы с библиотеками для парсинга данных.
